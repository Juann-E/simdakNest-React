untuk backend
    running-nya "npm start dev" atau pakai file bat nya
    1. masul ke backend
    2.  npm install -g @nestjs/cli
    3. nest ---version (untuk cek apakah sudah terinstall)
    4. nest new . --language ts
        (pakai npm saja)
    5. install dependency
        npm install @nestjs/jwt @nestjs/passport passport passport-jwt bcrypt
        npm install -D @types/passport-jwt @types/bcrypt
        npm install @nestjs/config @nestjs/typeorm typeorm mysql2
    6. npm install @nestjs/config
    7. npm install ts-node -D
    9. npm install @nestjs/mapped-types
    10. npm install class-validator class-transformer
    11. npm install exceljs
    ------
    12. npm install @nestjs/serve-static
    13. npm install --save-dev @types/express @types/multer
